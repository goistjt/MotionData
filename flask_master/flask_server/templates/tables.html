Server Endpoints

Endpoint Description
/ Dashboard/Sessions table
/tables Shows the API for the server
/getSessions/<device_id> Returns a list of all sessions that the current device has not contributed a record to
/getRecordRaw/<record_id> Download the raw data for the indicated record using a simple timestamp matching
/getRecordAnalyzed/<record_id> Download the fully analyzed/smoothed data for the indicated record
/getSessionRaw/<session_id> Download the raw data for the indicated session using an averaging for multiple records
/getSessionAnalyzed/<session_id> Download the fully analyzed/smoothed data for the indicated session
/createSession Post method that takes in record data of the given form and creates a new session {sess_desc: "", accelModels: [{time_val: long, x_val: float, y_val: float, z_val: float}], gyroModels: [{time_val: long, pitch_val: float, roll_val: float, yaw_val: float}], device_id: "", device_name: "", begin: long}
/addToSession Post method that takes in record data of the given form and adds the record to an existing session {accelModels: [{time_val: long, x_val: float, y_val: float, z_val: float}], gyroModels: [{time_val: long, pitch_val: float, roll_val: float, yaw_val: float}], device_id: "", device_name: "", sess_id: ""}
/deleteSession Delete method that takes in a session id and deletes the session {sess_id: ""}
